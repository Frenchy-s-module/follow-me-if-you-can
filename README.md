<h2 id="english">🇬🇧 English</h2>

## Overview
Follow Me If You Can is a powerful Foundry VTT module that enables tokens to follow other tokens on the map. Perfect for managing groups, NPCs, or creating dynamic chase sequences.

### Features
- 🎯 Simple one-click token following
- 🔄 Maintains following even after scene changes
- ⚡ Option for instant or animated movement
- 🎮 Intuitive UI with floating control buttons
- 🛡️ Prevents circular following patterns
- 🌐 Supports multiple languages

### Installation
1. In Foundry VTT's module manager, click "Install Module"
2. Enter the following URL: https://github.com/Frenchy-s-module/follow-me-if-you-can/releases/latest/download/module.json
3. Click "Install"

### Usage
1. Select the token you want to be the follower
2. Click the "Start Following" button
3. Select the target token to follow
4. Use the "Stop All" button to cancel all following relationships

### Settings
- **Keep Following After Scene Change**: Tokens continue following even after changing scenes
- **Instant Follow**: Choose between instant movement or smooth animation

### Support
For bug reports or feature requests, please [create an issue](https://github.com/Frenchy-s-module/follow-me-if-you-can/issues) on our GitHub repository.
