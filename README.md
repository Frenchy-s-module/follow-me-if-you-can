<h2 id="english">🇬🇧 English</h2>

## Overview
Follow Me If You Can is a powerful Foundry VTT module that enables tokens to follow other tokens on the map. Perfect for managing groups, NPCs, or creating dynamic chase sequences.

### Features
- 🎯 Simple one-click token following
- 🔄 Maintains following even after scene changes
- ⚡ Option for instant or animated movement
- 👥 Player access with permission-based system
- 🛠️ Selective control to stop specific followings
- 🎮 Intuitive UI with floating control buttons
- 🔘 Movable interface with reset position button for both GM and players
- 🎛️ Option to show or hide floating buttons
- 📋 Target selection dialog for players
- 🛡️ Prevents circular following patterns
- 🌐 Supports multiple languages
- 🔌 Robust socket system with auto-reconnection and error handling
- 💾 Client-side position saving with server sync for better performance

### Installation
1. In Foundry VTT's module manager, click "Install Module"
2. Enter the following URL: https://github.com/Frenchy-s-module/follow-me-if-you-can/releases/latest/download/module.json
3. Click "Install"

### Usage
1. Select the token you want to be the follower
2. Click the "Start Following" button
3. Select the target token to follow from the dialog
4. To stop following, GMs can click the "Stop All" button and select which following relationships to end
5. Drag the floating controls to reposition them, or click the Home button to reset their position

### Player Access
1. Enable the "Player Access" setting in the module settings
2. Players will see a follow button in their interface
3. Players can select a token they control and click the "Start Following" button
4. A dialog will appear allowing players to select which token to follow
5. Players can also move their control buttons and reset them to default position

### Settings
- **Keep Following Between Scenes**: When enabled, tokens will continue following their target after scene changes
- **Instant Following**: Tokens will move instantly to their target's position without animation
- **Allow Player Access**: Gives players access to following features for tokens they control
- **Show Floating Buttons**: Displays floating control buttons for easy access

- **シーン間で追跡を維持する**: 有効にすると、トークンはシーン変更後も対象を追跡し続けます

## 概述
Follow Me If You Can 是一个强大的 Foundry VTT 模块，使指示物能够在地图上跟随其他指示物。非常适合管理团队、NPC 或创建动态追逐序列。

### 功能
- 🎯 简单的一键式指示物跟踪系统
- 🔄 场景变更后仍保持跟踪
- ⚡ 即时或动画移动选项
- 👥 基于权限系统的玩家访问
- 🛠️ 选择性控制停止特定跟踪
- 🎮 带有浮动控制按钮的直观界面
- 🔘 可移动界面，GM 和玩家都有位置重置按钮
- 🎛️ 显示或隐藏浮动按钮的选项
- 📋 玩家的目标选择对话框
- 🛡️ 防止循环跟踪模式
- 🌐 支持多种语言
- 🔌 具有自动重连功能的强大套接字系统
- 💾 客户端位置保存与服务器同步

### 安装
1. 在 Foundry VTT 的模块管理器中，点击"安装模块"
2. 输入以下 URL：https://github.com/Frenchy-s-module/follow-me-if-you-can/releases/latest/download/module.json
3. 点击"安装"

### 使用方法
1. 选择您希望作为跟随者的指示物
2. 点击"开始跟随"按钮
3. 从对话框中选择要跟随的目标指示物
4. 要停止跟随，GM 可以点击"全部停止"按钮并选择要终止的跟随关系
5. 拖动浮动控件重新定位，或点击主页按钮重置其位置

### 玩家访问
1. 在模块设置中启用"玩家访问"设置
2. 玩家将在其界面中看到跟随按钮
3. 玩家可以选择他们控制的指示物并点击"开始跟随"按钮
4. 将出现一个对话框，允许玩家选择要跟随的指示物
5. 玩家还可以移动他们的控制按钮并将其重置为默认位置

### 设置
- **场景间保持跟随**：启用后，指示物将在场景变更后继续跟随其目标
- **即时跟随**：指示物将立即移动到其目标位置，无动画
- **允许玩家访问**：为玩家提供他们控制的指示物的跟随功能访问权限
- **显示浮动按钮**：显示浮动控制按钮以便于访问
